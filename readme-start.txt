Для развертывания проект необходимо:
1) Установить npm https://nodejs.org/en/download/
2) Установить gulp. В консоли данной директории открываем консоль и прописываем "npm install --global gulp";

npm install gulp
3) Далее прописываем "npm install", для установки всех модулей проекта;
4) Для компиляции прописываем "gulp default", после чего все изменения будут компилироваться после изменений;
